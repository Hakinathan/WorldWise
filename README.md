## Description

WorldWise is an React (with Vite.js) application. You can manage your past trips. For example, you can see your visited cities and countries.

## What I use ?

- React
- Vite.js
- React Router
  - React routes
  - React dynamic routes
  - React routes params
  - React NavLink
- React DOM
- Json Server (for fake API)
